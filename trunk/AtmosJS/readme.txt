
= Dependencies when running in a browser =
 * JavaScript 1.5 (used for maximum compatibility, see: http://en.wikipedia.org/wiki/JavaScript#Versions)
 * Some libraries from Crypto-JS (see the lib and lib-min folders)
  * crypto.js
  * hmac.js
  * sha1.js
  
= Dependencies when running in Node.js =
 * jsnode (npm install jsnode)
 * XMLHttpRequest (included in lib dir)
  
= Optional Components =
 * nodeunit (see nodeunit folder) - used for optional unit tests
